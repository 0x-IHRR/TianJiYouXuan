# 首页分层图片资产 v0

## 结论

首页后续采用 **Hybrid Layered Design**：图片负责真实材质、空间氛围和难以用 CSS 画好的物件；代码负责文字、导航、链接、视频、交互、遮罩和响应式。

不再把整页押在一张大背景图上，也不再用纯 CSS 画复杂静物。

## 实现原则

1. 图片只做氛围和材质，不承载核心文案。
2. 标题、导航、会员申请、社媒、表单、入口说明全部用 HTML。
3. 图片边界必须用 `mask-image`、渐变、透明度或混合模式融合，不做硬切卡片。
4. 视频区域使用图片窗洞 + 真实视频 + SVG/CSS mask。
5. 子页面优先复用这些层，而不是重新生成风格不一致的新图。

## 本轮生成方式

- 模式：内置 `imagegen`
- 参考：`images/草图/homepage-pc-continuous-space-direction.png`
- 目标：先生成首页可用的横向 16:9 真实质感切片。

生成的四类 prompt 分别对应：

- `hero-still-life`：右侧柜子花瓶静物，左侧留白。
- `film-window-frame`：真实灰泥窗洞，右侧留价值文案区域。
- `doorway-band`：三扇门、墙面箭头、地面光影。
- `table-invitation`：桌面、邀请函、碗、石头。
